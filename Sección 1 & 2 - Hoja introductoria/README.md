# Sección 1 & 2 - Hoja Introductoria

Esta carpeta contiene material de referencia y documentación introductoria para el curso de Angular.

## Contenido

- **angular-cheat-sheet.pdf**: Hoja de referencia rápida (cheat sheet) con los conceptos y sintaxis más importantes de Angular. Útil como material de consulta rápida durante el desarrollo.

## Propósito

Esta sección sirve como punto de partida del curso, proporcionando documentación de referencia que cualquier desarrollador principiante con la tecnología pueda consultar en cualquier momento durante su aprendizaje de Angular.
