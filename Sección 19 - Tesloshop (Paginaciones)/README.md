# Tesloshop - Con Paginación

Extensión de Tesloshop que añade **paginación** tanto en el backend como en el frontend para manejar grandes cantidades de productos de forma eficiente.

## Mejoras Implementadas

### Backend
- Endpoints de productos con paginación
- Query parameters: `limit`, `offset` o `page`
- Respuestas incluyen metadata de paginación (total, página actual, etc.)

### Frontend
- Componentes de paginación
- Navegación entre páginas
- Mantenimiento de estado de página
- Integración con servicios HTTP
